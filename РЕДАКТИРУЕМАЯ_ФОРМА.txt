ИНСТРУКЦИЯ ПО ДОБАВЛЕНИЮ РЕДАКТИРУЕМОЙ ФОРМЫ

1. ВЫПОЛНИТЕ НА ЛОКАЛЬНОЙ МАШИНЕ (cmd.exe):

git add .
git commit -m "Добавил редактируемые настройки формы обратной связи"
git push origin main

2. НА СЕРВЕРЕ ВЫПОЛНИТЕ:

cd /var/www/dron-site
source venv/bin/activate
git pull origin main
python manage.py makemigrations
python manage.py migrate
python manage.py collectstatic --noinput
sudo systemctl restart gunicorn
sudo systemctl reload nginx

ИЗМЕНЕНИЯ:
- Добавлена модель ContactFormSettings для редактирования формы через админку
- Теперь можно редактировать:
  * Заголовок формы
  * Описание формы  
  * Текст кнопки
- Все настройки доступны в админ панели в разделе "Настройки формы обратной связи"

ПОСЛЕ МИГРАЦИИ:
1. Зайдите в админ панель
2. Найдите раздел "Настройки формы обратной связи"
3. Отредактируйте заголовок, описание и текст кнопки
4. Сохраните изменения
5. Обновите сайт - изменения сразу отобразятся на форме
